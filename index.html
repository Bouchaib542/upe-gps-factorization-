Unified Prime Equation (UPE)

An explicit, windowed sieve that finds primes near any integer and produces Goldbach pairs with a tiny bounded correction. Paired with a GPS-inspired factorization method near √N.

Open Calculator Read the UPE paper

Minimal Window

Use a finite sieve up to P ≍ c₁·log N and a central window T ≍ c₂·(log N)². The first admissible offset usually hits directly (Δ_step = 0); otherwise ≤ 2.

Goldbach Mode

For E = 2x, search symmetric offsets t so that (x−t, x+t) are both prime. The sieve’s residue symmetry aligns both sides.

GPS Factorization

Anchor at √N, follow admissible tracks (wheel + CRT), and test near the predicted deviation. Escalate to a light QS-on-tracks if gaps are wide.

Unified Calculator

Enter large integers directly (digits only) or as powers like 10^1000. Uses BigInt and Miller–Rabin probable primality tests.
